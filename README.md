## :cd: Installation

```sh
git clone git@github.com:filipe1309/fc-hexarch-go-products.git
```

```sh
cd fc-hexarch-go-products
```

## :runner: Running

```sh
docker-compose up
```

> Access http://localhost:9000

## :white_check_mark: Tests

After up the container:

```sh
docker-compose exec app go test ./...
```
